About This Project

Komposta is a smart composting system that leverages technology to help farmers convert organic waste into compost efficiently, reducing waste and improving soil health.

Key Features

  • 🌱 Automated composting process tracking
  • 📊 Data-driven insights for soil improvement
  • ♻️ Waste-to-fertilizer conversion

Technologies

IoT Sensors, Mobile App (Kotlin), Firebase
